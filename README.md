# Open-Source-Project
## Aim: AI-Based Safety Equipment Detection using Computer Vision

## Project Overview:
This project focuses on detecting Personal Protective Equipment (PPE) such as helmets and safety vests in images or video streams using deep learning techniques. The goal is to automate safety monitoring in industrial environments.

## Objectives
1. Detect PPE (Helmet, Vest, etc.) in real-time
2. Improve workplace safety using AI
3. Build a scalable detection system using deep learning

## Tech Stack
1. Python
2. OpenCV
3. Deep Learning (YOLO / CNN)
4. NumPy, Pandas
5. Matplotlib
